# Bin2Dec

Projeto simples desenvolvido em C# com o objetivo de praticar lógica de programação, manipulação de strings, estruturas de repetição e conversão de números binários para decimal.

O projeto foi baseado no desafio proposto pelo repositório App Ideas:

- :contentReference[oaicite:0]{index=0}

---

## 📚 Objetivo

O programa permite que o usuário digite um número binário de até 8 dígitos e exibe seu equivalente decimal.

Além da conversão, o projeto também realiza validações para garantir que apenas números binários válidos (`0` e `1`) sejam inseridos.

---

## 🚀 Tecnologias utilizadas

- C#
- .NET
- Console Application

---

## 🧠 Conceitos praticados

- Estruturas de repetição (`for` e `foreach`)
- Condicionais (`if`)
- Manipulação de strings
- Conversão de tipos
- Potenciação com `Math.Pow()`
- Validação de entrada do usuário
- Lógica binária
